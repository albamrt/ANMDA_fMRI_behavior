# Instructions of usage:
1. Download this repository.
2. Create a folder named 'data' in the downloaded repository and place the data files 'behavior.csv'*, 'NMDAEvents.csv' and 'NMDAEventsProjecteAj-Treatment_DATA.csv' (last two downloaded from RedCap) in the folder.
3. Run the 'main.Rmd' file.
4. If wanted, upload the results back to GitHub.


* The file 'behavior.csv' can be created by using the script 'concat_behavior.py' on the cluster.
